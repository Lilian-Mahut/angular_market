## Run Locally

Clone the project

```bash
  git clone https://github.com/Lilian-Mahut/angular_pokedex.git
```

Go to the project directory

```bash
  cd angular_pokedex
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve
```
